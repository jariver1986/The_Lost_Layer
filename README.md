The Lost Layer Audio Controller es un proyecto escrito en Python que permite reproducir, detener y controlar el volumen de pistas de audio en tiempo real utilizando únicamente el teclado de la computadora.

Es una herramienta diseñada para productores musicales, DJs experimentales y desarrolladores que desean crear una interfaz personalizada para disparar stems (bass, drums, vocals, etc.) sin depender de un software DJ tradicional.

Este sistema funciona como un mini-sampler programable, capaz de mezclar múltiples pistas simultáneamente, asignar teclas específicas a cada acción y realizar ajustes suaves de volumen durante la reproducción.

🎧 Características principales

Reproducción de pistas de audio con una tecla (ej. Q = Bass).

Detención inmediata de audio (W = Stop).

Control de volumen progresivo desde el teclado (A = subir, S = bajar).

Permite mezclar varias pistas al mismo tiempo (según capacidad del sistema).

Funciona con archivos MP3 o WAV.

Código limpio, simple y fácil de ampliar.

Totalmente personalizable para agregar más pistas, efectos o controles.

🛠️ Tecnologías utilizadas

Python 3.x

pygame (motor de audio)

keyboard (detección de teclas en tiempo real)

🚀 Función del proyecto

Este proyecto nace de la idea de controlar stems de una canción o beat (bass, drums, vocals, synth, etc.) directamente desde Python para crear un sistema de performance musical experimental sin necesidad de una controladora DJ profesional.

Ideal para:

Live sets minimalistas

Experimentos sonoros

Artistas que quieran disparar capas de sonido

Productores que manejan stems separados

Usuarios que desean crear su propia herramienta de mezcla.

🧩 Extensión

El código está preparado para ampliarse fácilmente y soportar:

Más pistas y teclas asignadas

Efectos como fade-in, fade-out o delay

Integración con controladores MIDI (MPK Mini, Launchpad, etc.)

Integración con ESP32 (control vía serial)

Reproducción sincronizada por BPM
